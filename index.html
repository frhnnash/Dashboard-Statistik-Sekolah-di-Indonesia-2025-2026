Data untuk Pendidikan Indonesia Maju

📊 Tentang Dashboard
Dashboard Statistik Sekolah Indonesia 2025/2026 adalah platform analitik pendidikan yang menyajikan data komprehensif dari Dapodik (Data Pokok Pendidikan) Kemendikdasmen. Dashboard ini dirancang untuk membantu pengambil keputusan, peneliti, dan masyarakat umum dalam memahami kondisi pendidikan Indonesia secara data-driven.
📂 Sumber Data
Data bersumber dari Dapodik Kemendikdasmen, diperbarui per April 2026 untuk Tahun Ajaran 2025/2026. Mencakup data dari 38 provinsi di seluruh Indonesia, meliputi seluruh jenjang pendidikan dari SD, SMP, hingga SMA.

HTML5 · Tailwind CSS · Chart.js · JavaScript Vanilla
